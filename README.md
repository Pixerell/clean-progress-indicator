## Чистенький прогресс-индикатор


Созданный без сторонних библиотек всего с 3-мя файлами `app.js, index.html, index.css`

Индикатор умеет крутиться, плавно заполняться, прятаться и снова возвращаться. Для него сделан адаптив через `@media orientation`

Посмотреть можно по ссылке здесь - https://pixerell.github.io/clean-progress-indicator/

Или взять проект и запустить файлик index.html. В целом всё ;)
